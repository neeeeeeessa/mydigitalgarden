    HTML

  • Use < !DOCTYPE html > at the top of the page to indicate to the browser that this is HTML5. If not used, will most likely not change anything, but might cause issues sometimes.
    • Use < html > & < /html > to indicate the following code is HTML.
      • Use < head > & < /head > to include:
        • < title > Title placeholder < /title > to add page name on the tab.
        • < link rel="stylesheet" href="styles.css" /> & < link href="https://fonts.googleapis.com/..." rel="stylesheet" > to add css links.
        • < script src="index.js" > < /script > to add js links.
        • [tbd] Metadata for sharing in other websites (e.g. Twitter).
      • Use < body > & < /body > to include page elements:
          • Text elements
        • < h1 > & < /h1 > add title type 1.
        • < h2 > & < /h2 > add title type 2.
        • < h3 > & < /h3 > add title type 3.
        • < p > & < /p > add paragraph.
          • Interactive elements
        • < img src="" > include image. The image must be placed in the same folder as the coding file. Add width="100%" inside <> to resize image according to page size.
        • < button > & < /button > add pre-formated buttons.
        • < input type="text" > add input boxes. Add placeholder="" inside <> to include placeholder text.
        • < input type="password" > add input password boxes.
        • < a href="file.html" > Visible Text < /a > add a link to another html file.
        • < a href="website.com" target="_blank" > Visible Text < /a > add for an external website to open on a new tab.
          • List elements
        • Use < ol > & < /ol > and < ul > & < /ul > to start ordered and unordered lists.
          • < dl > & < /dl > add description lists.
            • < dt > & < /dt > add description titles.
            • < dd > & < /dd > add description details.
            • < li > & < /li > add list items.
